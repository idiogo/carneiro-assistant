# carneiro-assistant


## What
Whatsapp Business API integration that uses GPT-4 as engine. Audio files are transcribed by OpenAI Whisper before prompts to GPT.

## How

1. `npm install` the dependencies of package.json.

2. Create a .env file 

```
openaiApiKey=#####
whatsappFromNumber=######
whatsappAPIToken=######
```
